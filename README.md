# Coffee Machine Project (Python)

A simple command-line coffee machine simulator built using Python

### Features:
- Handles resources (water, milk, coffee)
- Lets you pick drinks and tracks remaining supplies
- Built entirely in Python  

## How to Run
Make sure you have Python 3 installed.

## Usage Example
buy
What would you like to drink? 1 - espresso, 2 - latte, 3 - cappuccino, back - to main menu:
I have enough resources, making you a coffee!
